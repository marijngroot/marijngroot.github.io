Maps

This section features some of the maps and applications which I made for my GIS course.

Remote sensing map
Google Earth Engine

This map shows an area in the Hautes-Alpes, France, surrounding the Lac de Serre-Ponçon. This is a lake in a valley, consisting mainly of melting water from the surrounding mountains. The different layers show the area in winter and summer, through several band combinations. This allows users to see the seasonal changes in snow coverage, vegetation, and the depth and size of the lake.

Fun fact: the lake is pictured in the background of this website!

Choropleth map · static
QGIS

This map shows the Sustainable Development Goal indicator 15.1.2; the proportion of Terrestrial Key Biodiversity Areas that is covered by protection regulations. The data were collected in 2017, the year in which the SDG indicators were developed. In the choropleth map, a darker colour indicates a higher percentage of protected biodiversity areas.

Choropleth map · interactive
ArcGIS Online

This web app features an interactive version of the choropleth map on biodiversity protection; instead of showing the data of a single year, it has separate layers with data from 2000 and from the latest year of measurements. By selecting the 'swipe' widget in the top right corner, users can swipe between the layers, and compare the improvements of different countries across two decades.

When the slider is on the far right, the data from 2000 is displayed. Setting it to the far left shows the map of the latest data.
